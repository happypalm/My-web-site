(필수태그)

html을 작성하기전에 <!doctype html>을 써야한다. 이는 이 문서가 html로 작성 되었음을 밝히는 문장이다.

모든 태그들은 <html> </html> 안에 써야한다

----------------------------------------------------------------------------------------
<head> </head> 안에 써야하는 태그

웹페이지 제목 : <title> </title>
글자를 깨지지 않게 함 : <meta charset="utf-8">
css를 쓰고 싶을 때 : <style> </style>

-----------------------------------------------------------------------------------------
<body> </body> 안에 써야하는 태그

<div> </div>
<span> </span>

강조표시 : <strong> </strong>

밑줄 : <u> </u>

제목에 주로 쓰이는 태그 : <h1>~<h6> </h1>~</h6>
(숫자가 작을 수록 크기가 줄고 두께가 얇아짐, h1~h6태그는 자동 줄바꿈이 된다.)

줄바꿈 : <br>
(뒤에 / 안붙이고 씀, 여러번 쓸 수 있음)

정해진 줄바꿈 : <p> </p>

이미지 넣기 : <img src="이미지 주소" width="숫자">
(width는 사진 넓이)

목록이나 목차 만들기 : <li> </li> - 자식태그
(부모 태그를 반드시 써야함)
목록 별로 나누고 싶을때 : <ul> </ul> - 부모태그
목록 앞에 숫자를 붙이고 싶을 때 : <ol> </ol> - 부모태그

특정 단어나 문장에 링크를 만들고 싶을 때 : <a href="링크주소"> </a>
링크를 새탭에서 열리게 하고 싶으면 <a href="링크주소" target="_blank"> </a>
링크가 무엇인지 부가설명 하고 싶으면 <a href="링크주소" title="부가설명"> </a>
